Un Progetto Open Source

QuizMaker è un progetto nato dalla passione per l'apprendimento e la tecnologia. È interamente ospitato su GitHub, il che significa che chiunque può vedere come è fatto, suggerire modifiche o aggiungere nuovi contenuti.

Come aggiungere nuovi Quiz

1

Crea il tuo quiz

Usa il nostro Editor Visuale per comporre le domande, le risposte e le spiegazioni. Non dimenticare di aggiungere un timer se vuoi rendere la sfida più avvincente!

2

Esporta il JSON

Una volta terminato, clicca su "Esporta Dataset JSON" nell'editor per scaricare il file sul tuo computer.

3

Invia una Pull Request

Vai sulla nostra repo GitHub, fai un fork del progetto e carica il tuo file nella cartella data/user_made/. Assicurati di aggiornare anche data/index.json per rendere il tuo quiz visibile agli altri!

Migliorare il Codice

Sei uno sviluppatore? Puoi aiutarci a migliorare la logica dell'app, il design o risolvere bug. L'app è costruita con Vanilla JavaScript e Tailwind CSS, il che la rende accessibile e facile da modificare.
